# GFP_paper用のスクリプト群

### requirements
* numpy
* pandas
* seaborn
* matplotlib
* scipy

### usage

* 任意のローカルフォルダにクローンしてください。

```bash
git clone git@github.com:HM-lab/for_GFP_paper.git
```

### note

* ファルダごとにreadmeを配置しているので確認して下さい。
* 'data_curation', 'statical_test'フォルダ以外は基本的にplot用のスクリプトです。
